git init создание локального репоззитория
git status посмотреть изменения статус
git add добавление файла в репозиторий
git config ввод имени и почты
git commit принять изменения
git log просмотреть историю коммитов
git branch создание ветки / branch посмотреть список / branch -d удалить ветку
git switch перемещение по веткам
git merge слияние веток
git remote add "" ссылка  связать удаленныйв репозиторий с локальным
git remote -v проверить добавился ли удаленный репозиторий
git push -u отправка данных в удаленный репозиторий
потом можно отправлять простой командой git push
git branch -r посмотреть список веток из удаленного репозиторрия
git fetch чтобы получить изменения внесенные в удаленный репозиторий 
git log чтобы сравнить изменения
git diff чттобы просмотреть различия в содержимом файлов
git pull получиь напрямую  в локальную ветку файлы из удаленного репозитория
